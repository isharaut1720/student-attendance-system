# 🧠 Student Attendance System

A simple and interactive **Student Attendance System** built using HTML, CSS, and JavaScript. This project allows users to manage student records and track attendance efficiently with date-wise data storage.

---

## 🚀 Features

- ➕ Add Students
- ✏️ Edit Student Name
- ❌ Delete Student
- 📅 Date-wise Attendance Tracking
- ✅ Mark Present / Absent
- 💾 Data stored using LocalStorage
- 🔍 Search Students
- 📊 Attendance Percentage Calculation
- 🎨 Color Indicator (Green / Yellow / Red)

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

---

## 📊 How It Works

- Students are stored globally
- Attendance is recorded date-wise
- Percentage is calculated dynamically based on total present days
